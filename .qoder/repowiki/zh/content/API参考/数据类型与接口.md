## 项目结构
OpenScreen 的类型体系主要分布在四类区域：
- 原生桥接层类型：位于 src/native/contracts.ts，定义了跨主进程与渲染进程通信的统一数据契约
- 视频编辑器内部类型：位于 src/components/video-editor/types.ts，定义编辑器内的区域、样式、动画等类型
- **新增**：DemoBuilder专用类型：位于 src/lib/demobuilder/types.ts，定义图文教程生成的核心数据结构
- 导出器配置类型：位于 src/lib/exporter/videoExporter.ts，定义导出阶段所需的参数集合

## 核心组件
本节对关键数据结构进行逐项说明，并标注字段含义、类型、是否可选及默认值。

### 原生桥接层核心类型
- 游标遥测点（CursorTelemetryPoint）
  - 字段
    - timeMs: number（毫秒；自录制开始的时间戳）
    - cx: number（屏幕空间 x 坐标）
    - cy: number（屏幕空间 y 坐标）
  - 约束
    - timeMs 单调递增
    - 坐标为绝对屏幕坐标
  - 使用场景
    - 遥测加载结果、播放器渲染、自动缩放建议
  - 参考路径
    - [src/native/contracts.ts:24-28](file://src/native/contracts.ts#L24-L28)
    - [src/components/video-editor/types.ts:171-186](file://src/components/video-editor/types.ts#L171-L186)

- 游标录制样本（CursorRecordingSample）
  - 继承自 CursorTelemetryPoint
  - 扩展字段
    - assetId?: string | null（资源标识）
    - visible?: boolean（可见性）
    - cursorType?: NativeCursorType | null（系统光标类型）
    - interactionType?: "move" | "click" | "mouseup"（交互事件）
  - 约束
    - interactionType 为有限集合
  - 参考路径
    - [src/native/contracts.ts:30-35](file://src/native/contracts.ts#L30-L35)

- 原生游标资产（NativeCursorAsset）
  - 字段
    - id: string
    - platform: NativePlatform
    - imageDataUrl: string（图像数据）
    - width/height: number（像素）
    - hotspotX/hotspotY: number（热点坐标）
    - scaleFactor?: number（缩放因子）
    - cursorType?: NativeCursorType | null
  - 约束
    - 宽高与热点需在图像范围内
  - 参考路径
    - [src/native/contracts.ts:37-47](file://src/native/contracts.ts#L37-L47)

- 游标录制数据（CursorRecordingData）
  - 字段
    - version: number（版本号）
    - provider: CursorProviderKind
    - samples: CursorRecordingSample[]
    - assets: NativeCursorAsset[]
  - 约束
    - samples/ assets 为空数组时仍需满足结构
  - 参考路径
    - [src/native/contracts.ts:49-54](file://src/native/contracts.ts#L49-L54)

- 系统能力（SystemCapabilities）
  - 字段
    - bridgeVersion: typeof NATIVE_BRIDGE_VERSION（桥接版本）
    - platform: NativePlatform
    - cursor: CursorCapabilities
    - project: { currentContext: boolean }
  - 约束
    - 版本号由常量统一管理
  - 参考路径
    - [src/native/contracts.ts:62-69](file://src/native/contracts.ts#L62-L69)

- 项目上下文（ProjectContext）
  - 字段
    - currentProjectPath: string | null
    - currentVideoPath: string | null
  - 约束
    - 两者可独立为 null
  - 参考路径
    - [src/native/contracts.ts:71-74](file://src/native/contracts.ts#L71-L74)

    - focusMode?: ZoomFocusMode
    - rotationPreset?: Rotation3DPreset
    - customScale?: number（1.0–5.0，两位小数）

    - focus 在 [0,1] 区间内

- 裁剪片段（TrimRegion）

- 注释区域（AnnotationRegion）

  - 约束

### DemoBuilder核心类型
- 截图数据（Screenshot）
  - 字段
    - id: string（唯一标识符）
    - url: string（本地文件路径或blob URL）
    - width/height: number（原始像素宽高）
    - order: number（项目中截图的排序索引）
    - originalName: string（用户上传时的原始文件名）
    - fileSize: number（文件大小，字节）
  - 约束
    - order 用于确定截图在教程中的显示顺序
    - url 支持本地文件路径和blob URL两种格式
  - 参考路径
    - [src/lib/demobuilder/types.ts:28-42](file://src/lib/demobuilder/types.ts#L28-L42)

- 热点标注（Hotspot）
  - 字段
    - id: string（唯一标识符）
    - stepId: string（所属步骤ID）
    - x/y/width/height: number（百分比坐标，相对于截图尺寸）
    - label: string（用户可见的标签/描述）
    - highlightStyle: HighlightStyle（高亮样式：border/background/pulse）
    - clickAnimation: ClickAnimation（点击动画：ripple/zoom/flash/none）
    - mouseTarget: Point | null（点击前光标移动的目标位置）
    - jumpToStepId: string | null（步骤跳转目标，null表示继续下一步）
    - tooltip?: string（浮动说明文本）
    - highlightColor?: string（自定义高亮颜色，十六进制）
    - shape?: HighlightShape（高亮区域形状：rect/circle/ellipse）
    - highlightDuration?: number（高亮显示时长，毫秒，默认1000）
  - 约束
    - x/y/width/height 为百分比值（0-100），相对截图尺寸
    - mouseTarget 为 null 时表示热点中心
    - highlightDuration 默认1000毫秒
  - 参考路径
    - [src/lib/demobuilder/types.ts:53-83](file://src/lib/demobuilder/types.ts#L53-L83)

- 鼠标动画（CursorAnimation）
  - 字段
    - type: CursorStyle（光标样式：default/hand/mac/windows/custom）
    - customIconUrl?: string（自定义PNG光标图像URL，仅当type为"custom"时）
    - startPosition: Point（起始位置，百分比）
    - endPosition: Point（结束位置，百分比）
    - movementType: MovementType（移动类型：linear/easing/bezier）
    - easingFunction?: string（GSAP缓动函数名称，仅当movementType为"easing"时）
    - bezierControlPoints?: BezierControlPoints（贝塞尔控制点，仅当movementType为"bezier"时）
    - movementDuration: number（移动持续时间，毫秒）
    - clickEffect: ClickEffect（点击效果：ripple/zoom/flash/none）
    - clickSound: boolean（是否播放点击音效）
    - delayBeforeClick: number（到达目标后点击延迟，毫秒）
  - 约束
    - position坐标为百分比值（0-100）
    - movementDuration 默认800毫秒
    - delayBeforeClick 默认200毫秒
  - 参考路径
    - [src/lib/demobuilder/types.ts:98-121](file://src/lib/demobuilder/types.ts#L98-L121)

- 转场效果（Transition）
  - 字段
    - type: TransitionType（转场类型：fade/slide-left/slide-right/slide-up/zoom/dissolve/wipe/none）
    - duration: number（转场持续时间，毫秒）
  - 约束
    - duration 默认500毫秒
  - 参考路径
    - [src/lib/demobuilder/types.ts:135-139](file://src/lib/demobuilder/types.ts#L135-L139)

- 字幕（Subtitle）
  - 字段
    - id: string（唯一标识符）
    - text: string（字幕文本内容）
    - start/end: number（相对步骤开始时间的开始/结束时刻，毫秒）
    - fontFamily: string（字体族）
    - fontSize: number（字号）
    - position: SubtitlePosition（垂直位置：top/center/bottom）
    - style: SubtitleStyle（字幕样式）
  - 约束
    - start < end
    - position 为垂直位置枚举
  - 参考路径
    - [src/lib/demobuilder/types.ts:153-166](file://src/lib/demobuilder/types.ts#L153-L166)

- 语音（Voice）
  - 字段
    - id: string（唯一标识符）
    - text: string（合成的文本）
    - audioUrl: string（生成的音频文件路径）
    - duration: number（音频时长，毫秒）
    - provider: VoiceProvider（TTS提供商：aliyun/openai/local）
    - voiceId: string（提供商内的声音标识）
    - speed: number（播放速度倍数，1.0为正常）
  - 约束
    - speed 为数值型倍速
  - 参考路径
    - [src/lib/demobuilder/types.ts:172-186](file://src/lib/demobuilder/types.ts#L172-L186)

- 步骤（Step）
  - 字段
    - id: string（唯一标识符）
    - screenshotId: string（显示截图的ID）
    - order: number（步骤排序索引）
    - title: string（步骤标题）
    - description: string（详细说明/操作指导）
    - hotspots: Hotspot[]（步骤截图上的热点标注）
    - cursor: CursorAnimation（光标动画配置）
    - subtitles: Subtitle[]（字幕覆盖）
    - voice: Voice | null（AI生成的语音旁白）
    - transition: Transition（移动到下一步的转场效果）
  - 约束
    - order 用于确定步骤执行顺序
    - voice 可以为null，表示无语音旁白
  - 参考路径
    - [src/lib/demobuilder/types.ts:190-210](file://src/lib/demobuilder/types.ts#L190-L210)

- 项目设置（ProjectSettings）
  - 字段
    - canvasWidth/Height: number（画布默认宽度/高度，像素）
    - aspectRatio: string（画布宽高比，如"16:9"）
    - background: DemoBackground（画布背景配置）
    - appearance: DemoAppearance（截图外观效果）
    - sound: DemoSound（声音设置）
    - defaultCursorType: CursorStyle（新步骤的默认光标样式）
    - defaultTransition: Transition（新步骤的默认转场）
    - defaultHighlightStyle: HighlightStyle（新热点的默认高亮样式）
    - exportSettings: ExportSettings（导出配置）
  - 约束
    - 默认值提供合理的初始配置
  - 参考路径
    - [src/lib/demobuilder/types.ts:259-279](file://src/lib/demobuilder/types.ts#L259-L279)

- 项目根（DemoProject）
  - 字段
    - version: typeof DEMO_PROJECT_VERSION（模式版本号）
    - id: string（项目唯一标识符）
    - name: string（项目名称）
    - description: string（项目描述）
    - createdAt/updatedAt: number（时间戳）
    - screenshots: Screenshot[]（所有可用截图）
    - steps: Step[]（有序的教程步骤列表）
    - settings: ProjectSettings（项目级默认值和导出配置）
  - 约束
    - version 用于向前兼容的迁移
    - 所有数组字段至少包含空数组

## 性能考量
- 游标采样
  - 遥测采样频率影响数据体积与播放器渲染开销，建议在保证体验的前提下降低采样率或压缩时间序列
- DemoBuilder项目管理
  - 截图导入时需要读取文件元数据，建议异步处理避免阻塞主线程
  - 大量截图的项目需要考虑内存占用和磁盘IO优化
- 区域计算
  - ZoomRegion 的 customScale 与旋转会增加合成成本，优先使用预设深度与轻量旋转
- 导出阶段
  - MotionBlur/Shadow/Blur 等效果叠加会显著提升编码压力，建议按需开启并限制强度
  - DemoBuilder的视频导出依赖Remotion框架，需要额外的安装和配置开销
- 内存占用
  - 大型注释/模糊区域与高分辨率导出会增加内存峰值，建议在导出前进行区域裁剪与降采样

## 结论
OpenScreen 的类型体系以原生桥接层为核心，向上提供统一契约，向下支撑编辑器、DemoBuilder和导出器的复杂业务。通过明确的字段定义、约束与默认值策略，系统在功能丰富与性能之间取得平衡。新增的DemoBuilder类型体系专门处理图文教程生成场景，与现有的视频编辑器类型形成互补，共同构成了完整的教程制作解决方案。建议在新增类型时遵循现有命名与约束风格，保持向后兼容与清晰的演进路径。
